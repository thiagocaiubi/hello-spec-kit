# Pydantic Validation API Contracts

This directory contains API contracts and schemas for the Pydantic validation feature.

## Files

- `health-api.yaml` - OpenAPI schema for health endpoint with Pydantic model
- `validation-errors.md` - Documentation of validation error responses

## Overview

All API schemas in this feature are automatically generated by FastAPI from Pydantic models. The contracts here document the expected behavior and structure for API consumers.
